## ✨ Key Features

### 🎨 Clean Corporate UI/UX Design
Built with a custom "Minimalist Clean" design. Crisp light-mode typography, corporate blue highlights, and highly structured spacing to give a premium, professional feel perfectly suited for enterprise roles (like WNS Global Services).

### 📄 One-Click PDF Resume Download
The "RESUME" buttons are natively hooked up to a beautifully formatted, auto-generated PDF version of Sahil's resume, allowing recruiters to download it instantly.

### ✨ Dynamic CSS Layouts
Rebuilt the project and experience sections using pristine CSS styling and layout animations (no generic thumbnail images required), making the text extremely scannable and polished.

### 🌌 Animated Gradient Effects
Features an incredibly smooth, uniquely animated geometric gradient mesh applied directly to the background layout, capturing attention immediately upon loading.

### 💼 Live Integrated Projects
Seamlessly showcases:
- **Student Management System** (Java, JSP, JDBC, MySQL)
- **Smart Bank Transaction System** (Java Collections)
- **Vehicle Service Management** (Core Java, OOPs)
